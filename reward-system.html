Reward tiers

Set the five point thresholds shown in the mobile app’s Rewards screen (from lowest to highest). Only this admin site can change these values or deduct users’ balances for redemptions. Sellers still earn points in the app when deals are completed; staff use Reward points to subtract points when a reward is given in person.

Tiers must be strictly increasing (each value greater than the previous). The app shows five claim buttons using these amounts.
